# Feedverse marketing site

This folder contains the marketing site + legal pages for Feedverse.

## Dev

```sh
cd website
npm run dev
```

## Build

```sh
cd website
npm run build
npm run preview
```

## Pages

- `/` (landing)
- `/legal/privacy.html`
- `/legal/terms.html`
- `/legal/data-deletion.html`
- `/legal/contact.html`

SEO files are in `public/` (served at site root): `robots.txt`, `sitemap.xml`.
